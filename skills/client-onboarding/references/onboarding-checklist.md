# Client Onboarding Checklist

Print and track for each new engagement.

## Pre-Kickoff

- [ ] Contract (MSA + SOW) fully signed by both parties
- [ ] NDA signed (if applicable)
- [ ] POPIA Data Processing Agreement signed (if handling personal data)
- [ ] Deposit invoice issued
- [ ] Deposit payment received
- [ ] Client contact details recorded (primary + technical + billing)
- [ ] Sales handoff notes reviewed
- [ ] Welcome pack prepared and sent
- [ ] Kickoff meeting scheduled (calendar invite sent)

## Kickoff Day

- [ ] Kickoff meeting completed
- [ ] Meeting minutes distributed within 24 hours
- [ ] Success criteria confirmed and documented
- [ ] Communication cadence agreed
- [ ] Escalation process confirmed

## Post-Kickoff (Within 3 Business Days)

- [ ] Paperclip project created
- [ ] Client access provisioned (see access matrix)
- [ ] Prudentia access received (repositories, environments, tools)
- [ ] Risk register created (minimum 3 risks identified)
- [ ] First sprint planned and committed
- [ ] Weekly status update schedule started
- [ ] Finance agent notified of billing schedule

## Access Matrix

| System | Client → Prudentia | Prudentia → Client |
|--------|-------------------|-------------------|
| Code repository | ☐ Invite received | ☐ Access granted |
| Dev/staging environment | ☐ Credentials received | N/A |
| Communication (Slack/Teams) | ☐ Workspace joined | ☐ Project channel created |
| Cloud console | ☐ IAM user created | N/A |
| Documentation | ☐ Shared folder access | ☐ Docs shared |
| Project board | N/A | ☐ Paperclip view access |

## Engagement Health Check (2 Weeks After Kickoff)

- [ ] Client has responded to at least 2 status updates
- [ ] First deliverable/demo has been completed
- [ ] No unresolved access issues
- [ ] Payment terms are being honoured (deposit cleared)
- [ ] Communication cadence is working (no missed updates)
- [ ] No scope misunderstandings surfaced

## Notes

- If any checklist item cannot be completed, escalate to CEO agent
- Do not start billable work until deposit is confirmed (except retainer clients)
- Keep a copy of this checklist in the Paperclip project for audit trail
