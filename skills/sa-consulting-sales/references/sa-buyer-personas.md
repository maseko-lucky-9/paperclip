# SA Buyer Persona Cards

Detailed persona cards for Prudentia Digital's target buyers in South Africa.

## Persona 1: SME IT Manager

| Field | Details |
|-------|---------|
| **Typical titles** | IT Manager, Technical Director, CTO (small company) |
| **Company size** | 10–50 employees |
| **Industry** | Professional services, retail, manufacturing, logistics |
| **Reports to** | MD / CEO / Founder |
| **Goals** | Modernise systems, reduce manual work, enable growth |
| **Daily frustrations** | Wearing too many hats, firefighting instead of building, no budget for senior hires |
| **Top objections** | "Budget is tight," "We've been burned by consultants before," "Can we do this in-house?" |
| **Evaluation criteria** | Trust, demonstrated competence, clear pricing, references from similar-sized companies |
| **Buying process** | 1-2 conversations, then MD approval. No formal procurement |
| **Best messaging** | "You get a senior engineer's full attention, not a revolving door of juniors" |
| **Budget cycle** | Rolling (no fixed cycle), but watch cash flow seasonality |

## Persona 2: Enterprise IT Director

| Field | Details |
|-------|---------|
| **Typical titles** | Head of IT, IT Director, VP Engineering, Chief Digital Officer |
| **Company size** | 100–5,000 employees |
| **Industry** | Financial services, insurance, mining, telecoms, large retail |
| **Reports to** | CIO / CFO / COO |
| **Goals** | Digital transformation, cloud migration, reduce technical debt, improve delivery speed |
| **Daily frustrations** | Legacy systems, slow procurement, vendor lock-in, internal politics, skills shortage |
| **Top objections** | "We need to go through procurement," "Need 3 quotes," "Preferred vendor list" |
| **Evaluation criteria** | Technical depth, security posture, compliance (B-BBEE, POPIA), enterprise references |
| **Buying process** | Technical evaluation → IT committee → Procurement → Legal → Sign-off. 6-12 weeks |
| **Best messaging** | "Enterprise-grade architecture and security, without the enterprise consulting price" |
| **Budget cycle** | Annual (usually aligns with financial year). Plan Q3-Q4, execute Q1-Q2 |

## Persona 3: Startup CTO / Technical Founder

| Field | Details |
|-------|---------|
| **Typical titles** | CTO, Technical Co-founder, Head of Product |
| **Company size** | 2–20 employees |
| **Industry** | Fintech, healthtech, edtech, proptech, agritech |
| **Reports to** | CEO / Board |
| **Goals** | Ship MVP fast, prove product-market fit, prepare for funding round |
| **Daily frustrations** | Too much to build, too few hands, technical decisions with long-term consequences |
| **Top objections** | "We're bootstrapping," "Can't you just use [no-code tool]?", "We'll hire full-time later" |
| **Evaluation criteria** | Speed, pragmatism, startup empathy, willingness to iterate |
| **Buying process** | 1 conversation, fast decision. Price sensitivity high but willing to pay for quality |
| **Best messaging** | "Production-grade from day one — no technical debt to pay off when you scale" |
| **Budget cycle** | Funding-round driven. Best time: post-raise or pre-fundraise (building the product to demo) |

## Persona 4: Government Procurement Officer

| Field | Details |
|-------|---------|
| **Typical titles** | Deputy Director: ICT, SCM Manager, Procurement Officer, Project Manager |
| **Company size** | Department / SOE / Municipality |
| **Industry** | Government (national, provincial, local), SOEs, government agencies |
| **Reports to** | Director, Chief Director, or DDG |
| **Goals** | Deliver on mandate, spend allocated budget, comply with regulations |
| **Daily frustrations** | Bureaucracy, political pressure, vendor non-delivery, procurement delays |
| **Top objections** | "Must go through SCM process," "Need minimum experience," "Price is not competitive" |
| **Evaluation criteria** | Formal: functionality score, price, B-BBEE. Informal: track record, references, reliability |
| **Buying process** | Tender published → Bid evaluation → Adjudication → Award. 2-6 months |
| **Best messaging** | "CSD-registered, Level 1 B-BBEE, technically qualified, and we deliver on time" |
| **Budget cycle** | April–March fiscal year. Budget allocated April–June. Spending pressure October–March |
